This is an AI-heavy port in C of Mirage Source 3.0.3 written originally in VB6, and I am calling it "Elysium" just because I like the name. The code follows similar structure and is very faithful to the original.

There are a lot of bugs but it works to a reasonable degree, but it is cross-platform at least and no .NET here (hate you, VB.net). The only dependency of the code is SFML (it's C-binding, to be exact). I'm updating it constantly.
